## Getting Started
1. 소스코드 클론 or ZIP 다운로드
2. (중략)
3. 재밌게 플레이

## Motivations
> 부트캠프 첫번째 프로젝트
> 
> 개발기간 5일 내 윈도우 콘솔에서 플레이할 수 있는 간단한 게임 만들기 프로젝트

- 처음 게임을 만들어보기 때문에 기획보단 구현 완성도에 집중해야 했음
- 기한이 제한적인만큼 개발 규모가 크지 않아야 했음
- 여유가 된다면 확장할 여지가 있는 적당히 도전적인 프로젝트여야 했음
